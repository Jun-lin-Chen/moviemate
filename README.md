# MovieMate
## ——A Custom Movie Recommendation System
![img.png](img.png)
## 系统说明

---
### 基础功能
1. 用户管理 + 用户画像 （基于搜索偏好，通过电影类别做）

2. 根据电影名字综合各个网站评分给出一个推荐值（真值）

3. 与网页GUI进行交互

### 拓展功能
1. 电影筛选

2. 评论区词云展示

3. 黑名单（屏蔽某一类电影/某一部电影 / 根据用户特点屏蔽（如未成年））

4. 根据用户画像推荐可能喜爱的电影

5. 待补充...
